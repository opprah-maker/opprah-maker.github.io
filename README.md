# Opprah Manyika : Aerospace Engineering Portfolio

Single-page engineering portfolio for **Opprah Manyika** (University Aerospace Engineering).

Source: <https://github.com/opprah-maker/opprah-maker.github.io>  
Live site: <https://opprah-maker.github.io/>

## Tech

- Static HTML5, CSS, vanilla JavaScript
- Template: [Miniport by HTML5 UP](http://html5up.net/miniport) (CCA 3.0 license)
- MathJax 3 for $\LaTeX$ equation rendering
- jQuery 3, Font Awesome 5, Responsive Tools

## Editing

The site is one file: `index.html`. The supporting CSS, JS, and webfonts live in `assets/`. Project images live in `images/`.

To add a new project, drop a representative image into `images/` (e.g. `pic08.png`) and copy an existing `<article class="box style2">` block in the `#portfolio` section.

To preview locally, just open `index.html` in a browser. GitHub Pages will serve the `main` branch at <https://opprah-maker.github.io/>.
